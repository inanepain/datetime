The `((Timescale))` *((enum))* simply defines three *((timestamps))* of various length: *((seconds))* (10 digits), *((milliseconds))* (13 digits) and *((microseconds))* (16 digits).

== Cases

* (((unit, seconds)))SECOND
    - 10 digits
* (((unit, milliseconds)))MILLISECOND
    - 13 digits
* (((unit, microseconds)))MICROSECOND
    - 16 digits
